## 🎨 Visual Design System

### Color Palette
```
Primary Background:    #0f0f0f (Dark)
Secondary Background:  #1a1a1a
Tertiary Background:   #252525
Borders:               #333333
Primary Text:          #ffffff
Secondary Text:        #b0b0b0
Tertiary Text:         #808080

Accents:
- Primary:   #000000 (Black)
- Success:   #10b981 (Green)
- Warning:   #f59e0b (Amber)
- Error:     #ef4444 (Red)
- Info:      #3b82f6 (Blue)
```

### Typography Scale
```
xs:   11px (0.688rem) — Labels, meta
sm:   12px (0.75rem)  — Captions
base: 13px (0.8125rem) — Body text
lg:   14px (0.875rem) — Labels, secondary
xl:   16px (1rem)     — Sections
2xl:  20px (1.25rem)  — Headings
3xl:  24px (1.5rem)   — Page titles
4xl:  32px (2rem)     — Hero titles
```

### Spacing (8pt Grid)
```
xs:  4px
sm:  8px
md:  12px
lg:  16px
xl:  24px
xxl: 32px
```

### Border Radius
```
xs:  4px   (small buttons, inputs)
sm:  6px   (badges)
md:  8px   (cards, containers)
lg:  12px  (modals, large cards)
xl:  16px  (section containers)
```

### Shadows
```
sm: 0 1px 2px 0 rgba(0, 0, 0, 0.05)
md: 0 4px 6px -1px rgba(0, 0, 0, 0.1)
lg: 0 10px 15px -3px rgba(0, 0, 0, 0.1)
xl: 0 20px 25px -5px rgba(0, 0, 0, 0.3)
```

### Animations
```
Fast:  150ms cubic-bezier(0.4, 0, 0.2, 1)
Base:  250ms cubic-bezier(0.4, 0, 0.2, 1)
Slow:  350ms cubic-bezier(0.4, 0, 0.2, 1)

All animations target 60 FPS
Reduce motion: respected via prefers-reduced-motion
```
