🚀 LANZAR EN 3 PASOS (15 MINUTOS)
═════════════════════════════════════════════════════════════════════════════════

  PASO 1: Get API Keys (10 min)
  ──────────────────────────────

  Key #1 — Clerk:
    → https://clerk.com
    → Create App
    → Copy: NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY + CLERK_SECRET_KEY

  Key #2 — Supabase:
    → https://supabase.com
    → Create Project (FREE)
    → Copy: NEXT_PUBLIC_SUPABASE_URL + SUPABASE_SERVICE_ROLE_KEY
    → SQL Editor: Run supabase/migrations/001_initial_schema.sql

  Key #3 — Vercel Blob:
    → https://vercel.com/dashboard
    → Storage → Blob → Create
    → Copy: BLOB_READ_WRITE_TOKEN


  PASO 2: Create .env.local (2 min)
  ─────────────────────────────────

  cat > .env.local << 'EOF'
  NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
  CLERK_SECRET_KEY=sk_test_...
  NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
  SUPABASE_SERVICE_ROLE_KEY=eyJhbGc...
  BLOB_READ_WRITE_TOKEN=vercel_blob_rw_...
  EOF


  PASO 3: Run (30 seconds)
  ──────────────────────

  npm run dev

  → http://localhost:3000 ✅
