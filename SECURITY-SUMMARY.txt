🔴 CRÍTICAS — ACCIÓN REQUERIDA ANTES DE DEPLOY:

  1. API KEYS EN PLAIN TEXT EN .env.local
     • OpenRouter, Anthropic, ElevenLabs, Clerk keys expuestas
     • Riesgo: Uso ilimitado de APIs, leak permanente si commiteado
     • Tiempo: 15 min | Acción: Rotar todas las keys

  2. CORS ABIERTO A CUALQUIER ORIGEN
     • /api/merge/versions permite cualquier dominio
     • Riesgo: CSRF, consumo de recursos no autorizado
     • Tiempo: 15 min | Acción: Whitelist de orígenes

  3. VALIDACIÓN DE INPUTS INSUFICIENTE
     • Sin validación de tamaño en /api/chat
     • Sin limpieza de prompt injection en generación
     • Riesgo: DoS, manipulación de respuestas de IA
     • Tiempo: 45 min | Acción: Validar all inputs

  4. ERROR MESSAGES EXPONEN DETALLES INTERNOS
     • 26 endpoints retornan error messages reveladores
     • Riesgo: Información de infraestructura leak
     • Tiempo: 45 min | Acción: Genéricos + logging seguro

  5. MISSING AUTHENTICATION EN /api/chat
     • Cualquiera puede llamar sin estar autenticado
     • Riesgo: Consumo económico ilimitado (~$1.50 por 100k tokens)
     • Tiempo: 30 min | Acción: Agregar auth + rate limit

  6. STRIPE WEBHOOK SIN IDEMPOTENCY
     • Replay attacks posible → múltiples pagos
     • Riesgo: Contabilidad desincronizada, créditos duplicados
     • Tiempo: 20 min | Acción: Verificar timestamp + ID tracking

  7. /api/webhooks GET SIN AUTENTICACIÓN
     • Info del webhook expuesta
     • Riesgo: Reconnaissance, enumeración de eventos
     • Tiempo: 15 min | Acción: Agregar auth

🔍 VERIFICACIÓN RÁPIDA POST-FIX:

  # 1. Auth en /api/chat
  curl -X POST http://localhost:3000/api/chat \
    -H "Content-Type: application/json" \
    -d '{"message": "test"}'
  → Debería retornar 401 Unauthorized ✅

  # 2. CORS cerrado
  curl -X OPTIONS http://localhost:3000/api/merge/versions \
    -H "Origin: http://evil.com"
  → NO debería incluir Access-Control-Allow-Origin ✅

  # 3. Error messages seguros
  curl http://localhost:3000/api/invalid
  → "An error occurred processing your request" (genérico) ✅
