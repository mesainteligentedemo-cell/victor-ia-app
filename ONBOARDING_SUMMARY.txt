TESTING
================================================================================

Para limpiar localStorage (mostrar tour nuevamente):

   En consola del navegador:
   localStorage.removeItem('victor_onboarding_done');
   location.reload();

Verificar que funciona:
   ✓ Tour aparece en primer acceso
   ✓ localStorage se llena
   ✓ Refresco no muestra tour
   ✓ ESC cierra
   ✓ Botones funcionan
   ✓ Indicadores clickeables
   ✓ Dark mode funciona
   ✓ Mobile responsive

================================================================================
PERSONALIZACIÓN
================================================================================

Cambiar pasos:
   Edita array onboardingSteps en Onboarding.tsx
   Cada paso necesita: id, title, description, hint, icon, color

Cambiar colores:
   Define variables CSS en tu raíz (layout.tsx):
   --bg: #ffffff
   --text: #1f2937
   --border: #e5e7eb
   --bg-secondary: #f3f4f6
   --text-secondary: #6b7280

Cambiar iconos:
   Importa diferentes iconos de lucide-react
   Asigna a la propiedad 'icon' de cada paso

Dark mode:
   Automático mediante prefers-color-scheme
   No requiere código adicional
