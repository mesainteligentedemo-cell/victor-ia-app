SECURITY LAYERS
================================================================================

Layer 1: HMAC-SHA256 Signature
  Protection: Payload integrity + Authentication
  Detection: Detects any tampering
  Implementation: createHmac('sha256', secret).update(payload).digest('hex')

Layer 2: Timestamp Validation
  Protection: Prevents old/stale requests
  Window: 5 minutes max age
  Clock Skew: ±30 seconds tolerance
  Detection: Rejects requests outside window

Layer 3: Cryptographic Nonce
  Protection: Prevents exact replay attacks
  Format: UUID v4 (collision probability < 1 in 5.3 billion)
  Store: In-memory Map or Redis
  Cleanup: Automatic expiry after 5 minutes

Layer 4: Timing-Safe Comparison
  Protection: Prevents timing attacks
  Method: Byte-by-byte constant-time comparison
  Implementation: Always compares all bytes regardless of position

================================================================================
ERROR CODES (7 SPECIFIC CODES)
================================================================================

400 - INVALID_JSON
      Request body is not valid JSON

400 - INVALID_STRUCTURE
      Missing required fields: payload, signature, nonce, timestamp

400 - INVALID_SIGNATURE
      Payload was tampered with or wrong secret

400 - TIMESTAMP_OUT_OF_RANGE
      Request is older than 5 minutes (prevents old requests)

400 - NONCE_REPLAY_DETECTED
      Same request sent twice (prevents replay attacks)

500 - WEBHOOK_SECRET_MISSING
      WEBHOOK_SECRET environment variable not configured

500 - INTERNAL_ERROR
      Unexpected server error during processing
