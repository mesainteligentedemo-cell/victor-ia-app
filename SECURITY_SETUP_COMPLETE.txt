================================================================================
PRÓXIMOS PASOS — ORDEN RECOMENDADO
================================================================================

PASO 1: LEER DOCUMENTACIÓN (10 minutos)
   Lee: ENV_SETUP_GUIDE.md (quick start)
   Lee: SECURITY.md (seguridad detallada)

PASO 2: PREPARAR DASHBOARDS (5 minutos)
   Abre 6 pestañas en navegador:
     1. https://dashboard.clerk.com
     2. https://dashboard.stripe.com
     3. https://app.supabase.com
     4. https://console.anthropic.com
     5. https://openrouter.ai
     6. https://elevenlabs.io

PASO 3: GENERAR SECRETOS LOCALES (5 minutos)
   PowerShell: openssl rand -base64 32 → JWT_SECRET
   PowerShell: openssl rand -base64 32 → ENCRYPTION_KEY
   Copiar en .env.local

PASO 4: COPIAR VALORES (15 minutos)
   Clerk → copiar 3 variables
   Stripe → copiar 3 variables
   Supabase → copiar 4 variables
   Anthropic → copiar 1 variable
   OpenRouter → copiar 1 variable
   ElevenLabs → copiar 3 variables

PASO 5: VALIDAR (2 minutos)
   Terminal: npm run validate-env
   Debería salir: ✅ TODAS LAS VARIABLES ESTÁN CONFIGURADAS

PASO 6: VERIFICAR COMPILACIÓN (5 minutos)
   Terminal: npm run build
   Verificar sin errores de env variables

PASO 7: INICIAR DEV (2 minutos)
   Terminal: npm run dev
   Abre: http://localhost:3000

PASO 8: EN VERCEL (cuando hagas deploy)
   Vercel Dashboard → Settings → Environment Variables
   Agregar TODAS las variables (privadas incluidas)
   Cambiar a live keys (pk_live_, sk_live_)

TOTAL: ~45 minutos desde cero
